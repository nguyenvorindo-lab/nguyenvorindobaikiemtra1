# Mini-project: Vẽ UI và tạo trang HTML

Dự án hoàn thiện trang web giới thiệu cấu trúc Bài 1 dựa trên bản phác thảo giao diện hệ thống.

## 📁 Cấu trúc thư mục dự án
- `index.html`: File chứa toàn bộ mã nguồn cấu trúc trang web theo bản phác thảo.
- `assets/ui-sketch.png`: Ảnh chụp bản phác thảo bố cục giao diện (UI Sketch).
- `README.md`: Tài liệu hướng dẫn và mô tả dự án.

## 🛠️ Phân vùng cấu trúc HTML
Trang web được triển khai chính xác theo bản vẽ thiết kế nằm trong thư mục `assets/` bao gồm:
1. `<header>` & `<nav>`: Tiêu đề hệ thống và thanh điều hướng nhanh.
2. `<main>`: Chứa 3 phân đoạn `<section>` độc lập (Tổng quan, Nội dung học, Biểu mẫu).
3. `<footer>`: Ghi nhận bản quyền thiết kế chính chủ bởi Nguyễn Võ Rin Đô.
